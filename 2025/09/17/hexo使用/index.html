windows 配置hexo

  1. 安装node.js(目前不理解是具体干嘛的)
  2. 安装cnpmnpm i -g cnpm --registry.npm.taobao.org
  3. 安装hexocnpm install -g hexo-cli
  4. 初始化目录
  1. 创建新文档hexo new "我的第一篇hexo博客"
  2. 将博客部署到github上
    • 首先创建一个新的仓库： 仓库名是固定的：必须是你的用户名.github.io
    • 回到windows终端安装一个插件cnpm install --save hexo-deployer-git
    • 修改配置文件_config.yml（就在博客目录下）,在最下方找到 Deployment,修改配置如下(注意：每个冒号后都要有空格)：
    • window执行npm install hexo-deployer-git --save
  3. 最后依次执行
